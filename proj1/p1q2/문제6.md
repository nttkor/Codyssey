# ✅ 문제 6. Pull Request 생성 및 반려 처리

1. 브랜치 푸시
```bash
git push origin add-menu
```

2. Pull Request 생성  
GitHub에서 Compare & pull request 클릭

PR 제목: 메뉴 화면추가 완료

내용: 이슈 #1 검토 요청 드립니다.

milestone: 메뉴판 추가 지정

생성 후 제출

3. 리뷰 의견 남기고 PR 반려  
PR의 Review 혹은 코멘트 탭에 의견 작성:

"녹차 메뉴만 버튼 사이즈가 이상하네요."

Close with comment 선택 → PR 반려
