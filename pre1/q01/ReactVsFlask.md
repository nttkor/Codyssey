# React와 Flask의 차이점

React와 Flask는 웹 개발에서 자주 사용되는 프레임워크지만 **역할이 다릅니다**.

> ? **Flask는 백엔드(서버 측)**  
> ? **React는 프론트엔드(화면 측)**

---

## ? 역할 비교

| 항목           | Flask                       | React                          |
|----------------|-----------------------------|---------------------------------|
| **역할**       | 백엔드 (서버 사이드)         | 프론트엔드 (클라이언트 사이드) |
| **언어**       | Python                      | JavaScript                     |
| **주요 기능**  | 서버 로직, API 처리, DB 연결 | UI 구성, 사용자 인터랙션       |
| **작동 위치**  | 서버에서 실행               | 브라우저에서 실행              |
| **출력**       | HTML, JSON, 데이터 응답      | 웹페이지 화면 구성             |

---

## ? 예시로 비교

- **Flask가 하는 일**
  - 로그인 처리
  - 회원가입 정보 데이터베이스 저장
  - `/api/user/123` 같은 REST API 응답
  - 이메일 전송 등 서버 작업 처리

- **React가 하는 일**
  - 버튼 클릭 시 화면 전환
  - 입력창 내용 실시간 반영
  - 서버로부터 받은 데이터를 테이블로 표시
  - UI 애니메이션, 팝업 등 사용자 인터페이스 제공

---

## ? 함께 사용하는 구조

대부분의 현대 웹 프로젝트는 다음과 같은 구조를 사용합니다:

- **Flask**: 서버에서 API를 제공하고 데이터베이스와 상호작용
- **React**: 사용자에게 보여지는 화면을 구성하고, Flask API와 통신
